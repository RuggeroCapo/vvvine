# Amazon Vine Efficiency Chrome Extension - Project Memory

## Project Overview
**Goal**: Create a highly efficient Chrome extension for Amazon Vine that prioritizes speed and productivity over visual aesthetics.

## Target Page
- **URL Pattern**: Amazon Vine product listing pages with pagination
- **Main Container**: `#vvp-items-grid`
- **Item Container**: `.vvp-item-tile`
- **Key Data Attributes**:
  - `data-asin`: Product identifier
  - `data-recommendation-id`: Unique recommendation ID
  - `data-img-url`: Product image URL

## Technical Architecture

### Manifest V3 Structure ‚úÖ
```json
{
  "manifest_version": 3,
  "name": "Amazon Vine Efficiency Enhancer",
  "content_scripts": [{
    "matches": ["*://www.amazon.*/vine/*"],
    "js": ["content.js"],
    "css": ["styles.css"]
  }],
  "permissions": ["storage", "activeTab"]
}
```

### Data Storage Strategy ‚úÖ
- **Chrome Local Storage**: Store seen ASINs, user preferences
- **Session Storage**: Current page filters, sort settings
- **Memory**: Current page product data for fast filtering

### Performance Requirements ‚úÖ
- **Fast Loading**: Content script must not delay page rendering
- **Memory Efficient**: Limit stored data, cleanup old entries
- **Responsive UI**: All interactions should feel instant (<100ms)
